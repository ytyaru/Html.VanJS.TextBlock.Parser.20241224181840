# 括弧の英語名

記号|英語|日本語
----|----|------
-|bracket|括弧
`()`|`round bracket`/`paren`|丸括弧
`[]`|`bracket`/`square bracket`/`box bracket`|角括弧
`{}`|`brace`/`braces`/`curly bracket`/`curl`|波括弧
`｢｣`|`corner bracket`|鍵括弧
`〔〕`|`tortoise shell bracket`|亀甲括弧
`<>`|`angle bracket`|山括弧
`【】`|`lenticular bracket`|隅付き括弧

* `｛［（　）］｝`の順が世界的に多数派である
